Contenedores y estilos

En html la mayoría de los elementos están categorizados como elementos de bloque o elementos de línea.

Los elementos de bloque son aquellos que comienzan en una nueva línea, como por ejemplo:

  • <h1>
  • <form>
  • <li>
  • <ul>
  • <ol>
  • <table>
  • <div>
  • Entre otros...

Recordemos que una lista no se puede centrar normalmente. Para esto se requiere hacer uso de CSS pero eso lo haremos más adelante

Los elementos de línea son aquellos que se visualizan sín líneas de quiebre, como por ejemplo:

  • <b>
  • <a>
  • <strong>
  • <img>
  • Entre otros...

El elemento <div> es un elemento de bloque que suele utilizarse como contenedor para otros elementos html.

Los contenedores son etiquetas que se utilizan para identar y dar estilo a los textos.

A los contenedores se les puede aplicar estilos para modificar la apariencia del contenido de todo un bloque. Ya lo vimos, por ejemplo, para el caso de una imagen.

Para modificar el estilo de un bloque, como podrás revisar en la fuente de este fichero, se utiliza el atributo style acompañado de los valores que cambian el estilo, algunos de estos valores pueden ser:

  • background-color:(color), que cambia el color de fondo del bloque.
  • color:(color), que cambia el color del texto del bloque.
  • padding:(px), que define el grosor del bloque.
  • padding-bottom:(px)
  • padding-left:(px)
    •
  • padding-right:(px)
    •
  • padding-top, entre otros...
    •

Un párrafo, a la derecha y con fondo de color y con letras de color verde, se definiría de la siguiente forma:

<div align="right" style="background-color:#09F7BB;color:green;padding:10px">
<p> Este es un párrafo dentro del contenedor div, a la derecha, con fondo de color y letras de color verde. </p> </div>

Así como existen contenedores de bloque, también los hay de línea, como es el caso del contenedor <span>

<h3> En esta frase hay una palabra de color <span style="color:red"> rojo </span></h3>

Como ya lo hemos mencionado a lo largo del curso, más adelante vamos a produndizar con los estilos y contenedores a través de CSS.

Existen otros elementos categorizados como elementos de bloque, aunque se ocupan algo menos. Éstos son:

  • APPLET, embebido en los aplet Java.
  • IFRAME, inline frame.
  • INS, texto insertado.
  • MAP, imagen map.
  • SCRIPT, script dentro de un documento HTML.

Los elementos de línea pueden insertarse dentro de elementos de bloque, pero un elemento de línea no puede contener elementos de bloques.

Otros contenedores, muy comunes, son:

  • <blockquote> - Texto con identación.
  • <address> - Permite escribir información de contacto.
  • <note> - Permite escribir una nota.
  • <fn> - Crea una nota al pie.
  • <banner> - Permite que un bloque de texto adopte una posición fija en relación a la página.
  • <pre> - Permite escribir texto preformateado; manteniendo espacios, saltos de líneas y tabuladores intactos.
