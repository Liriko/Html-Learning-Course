Formularios

Un formulario en html sirve para poder capturar información del usuario. El atributo action llevará al usuario a otro destino cuando envíe su información al formulario a través del submit.

El atributo method nos indica el método http que se utilizará para enviar la información del formulario de un lado a otro, cuando el usuario presione el botón submit. Los valores para este atributo pueden ser GET o POST. Cuando se usa GET la información del formulario podrá visualizarse en la URL. Luego, es recomendable usar POST cuando se desea actualizar información sensible, ya que ofrece una mejor seguridad al no desplegar estos datos en la URL, a diferencia del método GET.

Para capturar la información del usuario se hará uso de los elementos del formulario, como campos de texto. El elemento <input> tiene muchas variaciones, dependiendo del tipo de atributo. Puede ser un texto, un password, radio, url, submit, etc... La etiqueta <input> no tiene etiqueta de cierre.

Si seleccionamos el tipo radio, haremos que el usuario deba escoger una de varias opciones.

Si seleccionamos el tipo checkbox, le damos la oportunidad al usuario de poder escoger entre una y más opciones.

Finalmente existe el elemento o botón Submit que gatillará el envío de la información del formulario a dónde corresponda y a través el método de envío definido en el formulario.
