Imágenes - Etiqueta IMG

Esta etiqueta solo tiene atributos y es útil para agregar imágenes a la página.

A continuación, se muestra un ejemplo de cómo se utiliza esta etiqueta, en donde se llamará a una imagen desde la web.

<img src="https://cdn.icon-icons.com/icons2/1465/PNG/512/756exclamationmark_100528.png" alt="Exclamación" width="150px" height="150px" />

Ahora, expliquemos los atributos utilizados:

El atributo src es el source, o "fuente", en español, de la imagen. Ésta puede ser una url o una ruta física.

Cuando se utiliza una ruta física, la raíz nace desde la carpeta donde está ubicada la página. Es decir, si por ejemplo vas a utilizar una imagen llamada img.png, ubicada en la misma carpeta de la página, la ruta quedaría así:

<img src="img.png" />

Ahora bien, si la misma imagen estuviese dentro de otra carpeta, dentro de la raíz, como por ejemplo, una carpeta img, entonces quedaría así:

<img src="img\img.png" />

El atributo alt se utiliza para darle un texto alternativo a la imagen, en caso que ésta no aparezca por cualquier motivo.

Observación: Usar imagenes muy grandes puede volver lenta la página.

Ahora, no puede alinearse la imagen como se hace con un párrafo o texto, porque no aplica de la misma forma. Para ello, la imagen debe encontrarse en un contenedor y la alineación debe aplicarse a ese contenedor. Los contenedores los explicaré más adalante. Pero para efectos de este ejemplo, vamos a usar un contenedor Div.
