Atributo Width

El atributo width es un atributo de dimensionamiento, y ayuda a re-dimensionar ciertos elementos. Por ejemplo, para que una línea horizontal incremente su ancho a 50 pixeles, se utilizaría el atributo de la siguiente forma:

<hr width="50px" />

Así como existe un atributo para re-dimensionar ancho, existe uno para re-dimensionar el alto, y funciona de la misma manera. Estamos hablando de la atiqueta height. ¡Haz la prueba!
