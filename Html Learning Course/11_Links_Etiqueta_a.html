Links - La etiqueta a

La etiqueta <a> es útil para dejar hypervínculos a otra páginas o imágenes. El atributo <href> define el link de destino al que redirige el hipervínculo.

A continuación, un ejemplo de cómo hacer uso de esta etiqueta, creando un hipervínculo que nos re-direccione al home de Google.

<a href="https://www.google.cl/"> Aquí va el mensaje que será clicable por el usuario. </a>

Y a continuación se muestra como se vería el hypervínculo:

Aquí va el mensaje que será clicable por el usuario.

Ahora, para que el mismo hypervínculo corresponda a una imagen, solo debe anidarse la etiqueta <img> dentro de la etiqueta <a>. De la siguiente forma:

<a href="https://www.google.cl/"><img src="https://upload.wikimedia.org/wikipedia/commons/5/51/Google.png" height="50px" width="200px"></img></a>

Pero tenemos otro atributo importante a considerar, el atributo target, que especifica en dónde se visualizará el nuevo documento. Si al atributo target le damos el valor _blank le estaremos indicando al navegador que debe abrir una nueva pestaña o ventana para mostrarnos el nuevo contenido. Por el contrario, si usamos el valor _self, por ejemplo, le indicaremos que debe mostrarnos la nueva página en la misma pestaña, actualizando todo el contenido.
