Atributo Align

El atributo align sirve para alinear algunos elementos. Los atributos complementan a las etiquetas. El atributo align, por ejemplo, puede acompañar a una etiqueta de párrafo, de esta forma:

<p align="center"> Texto que estará alineado. </p>

Este párrafo está alineado a la izquierda. Se utilizó el atributo align="left"

Este párrafo está alineado al centro. Se utilizó el atributo align="center"

Este párrafo está alineado a la derecha. Se utilizó el atributo align="rigth"
