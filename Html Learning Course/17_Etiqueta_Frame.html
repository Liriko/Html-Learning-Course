La etiqueta Frame

Una página puede dividirse en marcos utilizando un marco especial de documento.

La etiqueta <frame> define una ventana o marco específico dentro de un <frameset>.

Cada <frame> en un <frameset> puede tener distintos atributos, como lo son los bordes, scroll, la habilidad de re-dimensionarse, etc.

El elemento <frameset> especifica un número de columnas o filas en un frameset, asi como el porcentaje o número de pixeles de espacio que ocupa cada una.

Ojo, la etiqueta <frameset> no está soportada en HTML5.

El atributo <noresize> especifica que ningún usuario podrá redimensionar el elemento <frame>.

El contenido de un frame debe definirse haciendo uso del atributo src.

El atributo <noframes> le posibilita a los navegadores que no soportan frames a mostrar la página. Este elemento puede contener una página alternativa u otro elemento.
