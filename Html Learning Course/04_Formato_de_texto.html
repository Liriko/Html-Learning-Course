Formatos de texto

Esto es un texto regular. Corresponde a la etiqueta <p>, de párrafo.

Esto es un texto en negrita. Se logra con la etiqueta <b>.

Esto es un texto en cursiva. Se logra con la etiqueta <i>.

Esto es un texto importante. Se logra con la etiqueta <em>.

Esto es un texto importante. Se logra con la etiqueta <strong>.

Esto es un texto grande. Se logra con la etiqueta <big>.

Esto es un texto pequeño. Se logra con la etiqueta <small>.

Esto es un texto insertado. Se logra con la etiqueta <ins>.

Esto es un texto de subíndice. Se logra con la etiqueta <sub>.

Esto es un texto de superíndice. Se logra con la etiqueta <sup>.

Esto es un texto eliminado. Se logra con la etiqueta <del>.

Las etiquetas <b> y <strong>, <i> y <em> podrían parecerte lo mismo, pero no lo son. La etiqueta <strong> es una etiqueta de frase y define un texto importante. Ocurre lo mismo con la etiqueta <em>.
